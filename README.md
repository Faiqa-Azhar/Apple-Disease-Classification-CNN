# Apple-Disease-Classification-CNN
Deep Learning based apple disease classification using CNN

Project Description:
This project uses Convolutional Neural Networks (CNN) to classify apple leaf diseases using labeled image datasets. The model was trained and evaluated using Deep Learning techniques and tested on real-world images for prediction analysis.

Features:
- Image Classification using CNN
- Model Training & Evaluation
- Accuracy & Loss Visualization
- Underfitting & Overfitting Analysis
- Real-time Prediction Testing

Technologies Used:
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Matplotlib
- Scikit-learn
